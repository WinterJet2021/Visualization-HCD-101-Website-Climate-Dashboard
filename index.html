This dashboard visualizes key climate data to help understand the scope and impact of climate change. Explore temperature trends, emissions by country, and analysis of climate policy language.

Climate Policy Language Analysis

Analyze the most common terminology used in climate policy documents. This visualization shows the frequency of key terms in different policy contexts.

Key Insights:

  • Terms like "mitigation" and "adaptation" are increasingly common
  • Emphasis has shifted from "global warming" to "climate change" over time
  • Economic terms appear more frequently in newer policy documents

About This Project

This visualization dashboard was created as an individual project for my data visualization course. It uses D3.js to create interactive visualizations of climate data.

Data Sources:

  • Temperature data: NASA GISS Surface Temperature Analysis
  • Emissions data: Our World in Data, based on the Global Carbon Project
  • Policy text: Analysis of publicly available climate policy documents

Design Process:

The visualizations were designed with the following considerations in mind:

  • Accessibility: Color schemes chosen for colorblind accessibility
  • Interactivity: Controls allow users to explore different aspects of the data
  • Clarity: Each visualization focuses on communicating one key message

Design Evaluation:

After creating the initial dashboard, I gathered feedback from peers and made the following improvements:

  • Enhanced color contrast for better readability
  • Added tooltips to provide more detailed information
  • Included "Key Insights" sections to highlight important findings
  • Made the visualizations responsive to different screen sizes

Project Reflection:

Creating this dashboard taught me several key lessons about effective data visualization:

  • Understanding the audience is crucial for creating accessible visualizations
  • Interactive elements should enhance understanding, not distract from the main message
  • Simplicity is often more effective than complexity in conveying information
  • Data visualizations should tell a coherent story to engage users
